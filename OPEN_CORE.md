# Open Core Model

This repository contains the **open-source core** of the MES platform.

## License

## Open Core Structure
The MES platform follows an **Open Core** model:
- This repository provides the core, extensible functionality.
- Advanced, enterprise, or commercial features are **not included here**.
- Enterprise features are distributed under separate proprietary licenses.
